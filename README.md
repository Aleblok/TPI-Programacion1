# Trabajo Práctico Integrador - Programación 1

## Tema
Gestión de Datos de Países en Python: filtros, ordenamientos y estadísticas.

## Descripción
Este proyecto permite gestionar información de países utilizando Python.  
El programa lee datos desde un archivo CSV y trabaja con una lista de diccionarios para realizar consultas, filtros, ordenamientos y estadísticas básicas.

## Archivos del proyecto
- `main.py`: código fuente principal del programa.
- `paises.csv`: archivo CSV con el dataset base de países.
- `README.md`: descripción del proyecto e instrucciones de uso.

## Estructura de datos utilizada
Cada país se representa mediante un diccionario con las claves:
- nombre
- poblacion
- superficie
- continente

Todos los países se almacenan dentro de una lista.
